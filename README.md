# StreamWise – SQL-Powered Movie Recommendation Analytics Platform

## Overview

StreamWise is a database-driven movie recommendation analytics platform built using PostgreSQL, Python, SQLAlchemy, and Streamlit.

Unlike traditional movie recommendation demos, StreamWise focuses on answering the central business question:

> **"For a given user, what are the best unseen movies to recommend next, and why?"**

The project demonstrates database design, SQL query optimization, recommendation logic, and interactive analytics using the MovieLens 100K dataset.

---

## Objectives

* Design a normalized relational database for a streaming platform.
* Implement multiple SQL-based recommendation strategies.
* Ensure every recommendation includes a relevance score and an explanation.
* Build an interactive dashboard to explore recommendations and analytics.
* Demonstrate explainable recommendation systems without using machine learning.

## Recommendation Engines

### 1. Genre-Based Recommendation

Recommends highly rated unseen movies belonging to genres preferred by the selected user.

**Uses**

* Ratings
* Genres
* Average movie ratings

---

### 2. Because You Watched

Generates recommendations by comparing the genres of movies the user previously rated highly.

Features:

* Excludes previously watched movies
* Weighted recommendation score
* Dynamic explanation for every recommendation

Example:

> Recommended because you highly rate Drama, Romance movies.

---

### 3. Similar Users Recommendation

Identifies users with similar viewing behaviour and recommends movies they enjoyed.

Features:

* User similarity detection
* Community support score
* Dynamic recommendation explanation

Example:

> Recommended because 261 users with similar tastes enjoyed this movie (Average Rating: 4.47/5).
